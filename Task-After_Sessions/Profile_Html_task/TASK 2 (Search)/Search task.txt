ده يا سيدى بيتم من خلال 6 steps يلا بينا 
step 1 : المتصفح يبدأ بقراءة ملف الـ HTML من الأعلى إلى الأسفل.

يقوم ببناء شجرة DOM (Document Object Model)، وهي تمثيل لكل عناصر الصفحة.

step 2: شبه الخطوة اللى فاتت وهو انه تحليل CSS → إنشاء شجرة CSSOM
بعد ذلك، يتم تحليل ملفات وأنماط CSS.

يُنشئ المتصفح شجرة CSSOM (CSS Object Model)، وهي تمثل الشكل النهائي لكل عنصر (الألوان، الخطوط، المسافات...).

setp 3 : دمج DOM وCSSOM → شجرة العرض (Render Tree)
المتصفح يدمج شجرتي DOM وCSSOM لإنشاء شجرة العرض التي تحتوي فقط على العناصر المرئية. وكده تبقى 
جاهزة للخطوة رقم 4 

step 4 : Layout / Reflow  وهنا بحسي حجم كل عنصر على الشاشه واشوف هيتحط فى انهو postion
وباللون كذا وال font-size كذا من الاخر كانى بعمله (ID) او بطاقة هوية
لكل عنصر عندى

step 5 : خططنا وحددنا كل عنصر فين وحجم ازى نبدا بقى Painting وده اللى هيبدا يظهر لل 
user على ال screen

step 6 : الخطوة الاخيرة وهى Compositing او التجميع ودى بدمج فيها 
كل ال cmpounds مع بعض علشان يظهرلى موقع متناسق والدنيا تبقى تمام

 ملحوظة:
لو كانت هناك JavaScript، يمكنها تعديل DOM أو CSSOM أثناء التحميل أو بعده، مما قد يُعيد خطوات التخطيط والرسم.

ما هو الـ DOCTYPE وما فائدته؟ 

هو بيقى سطر فى اول صفحة ال html بيعرف ال المتصفح انى شفال باحدث اصدار
من ال html اللى هى HTML5 ودى يعتبر مجمعه كل Standards Mode
وال tags اللى فى الversions القديمة 
لو لم تكتب الـ DOCTYPE، قد يعمل المتصفح في وضع قديم (Quirks Mode) مما يؤدي إلى أخطاء في العرض والتصميم. 

وتكتب بهذا الشكل 

<!DOCTYPE html>
وده اسهل واوضح ويعتبر friendly اكثر من  أمثلة قديمة (HTML4 أو XHTML) زى

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"
  "http://www.w3.org/TR/html4/loose.dtd">
